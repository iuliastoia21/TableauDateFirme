# Business Intelligence Project in Tableau

This project was developed in Tableau and focuses on analyzing company data from Sibiu using structured business information and visual indicators.
The dashboard allows the analysis of companies based on economic and business-related indicators, helping to identify patterns and present information in a clear and interactive way.
## Tools used
- Tableau
- Data visualization
- Business Intelligence
- Data analysis

## Skills developed
- Working with structured data
- Creating interactive dashboards
- Data visualization
- Analytical thinking
- Attention to detail
